Project Title: Hydrological Data Monitoring and Analysis Platform

Description: The Hydrological Data Monitoring and Analysis Platform is a user-friendly set of tools designed to support better water resource management through data-based understanding. It helps collect, view, and study hydrological data in an organized way. The platform includes the WAIN Web App and the DAM-IN Web App, which allow users to explore data interactively and carry out detailed hydrological analysis. This platform aims to assist researchers, policymakers, and environmental experts in making well-informed decisions and managing water resources more effectively.
